# cipherchat

Проект для обучения LoRA над Qwen2.5-7B-Instruct на задачах шифрования/дешифрования
(Caesar, Vigenere, Substitution) для английского, словацкого и украинского.

Структура:
- config/                — конфиги тренировки
- data/raw/              — сырые данные (старый датасет и т.п.)
- data/processed/        — подготовленные jsonl для обучения/валидации
- src/data/              — генерация и конвертация датасетов
- src/model/             — обучение LoRA и инференс
- scripts/               — bash-скрипты для подготовки данных и тренировки
